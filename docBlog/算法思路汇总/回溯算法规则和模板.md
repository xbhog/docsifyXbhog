# 回溯算法

## 回溯规则：

**组合是不强调元素顺序的，排列是强调元素顺序**。

**集合的大小就构成了树的宽度，递归的深度，都构成的树的深度**。

回溯三部曲：

1. 回溯函数模板返回值以及参数：其函数返回值一般为void

2. 回溯函数终止条件

   ```java
   if (终止条件) {
       存放结果;
       return;
   }
   ```

3. 回溯搜索的遍历过程

   ```java
   for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
       处理节点;
       backtracking(路径，选择列表); // 递归
       回溯，撤销处理结果
   }
   ```

完整模板：

```java
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```
