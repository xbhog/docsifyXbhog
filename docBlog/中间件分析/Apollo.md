## Apollo架构及优势

### Apollo优势

1. 能够集中化管理应用不同环境、不同集群的配置;
2. 配置修改后能够实时推送到应用端;
3. 具备规范的权限、流程治理等特性，适用于微服务配置管理场景;
4. 适配主流语言(`Java .net Golang Python Node PHP C Http API`)等;
